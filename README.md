# AtlanticaPadel Club Manager (APCM)

## Descrição

O AtlanticaPadel Club Manager é uma plataforma web desenvolvida no âmbito do Projeto Final da Licenciatura em Gestão de Sistemas e Computação da Universidade Atlântica.

O objetivo é disponibilizar uma solução integrada para a gestão de academias de pádel, permitindo administrar reservas de campos, sócios, treinadores, aulas e indicadores de gestão.

---

## Objetivos

- Gestão de reservas
- Gestão de sócios
- Gestão de treinadores
- Gestão de aulas
- Dashboard administrativo
