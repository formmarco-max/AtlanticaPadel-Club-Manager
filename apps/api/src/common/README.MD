# Common

A pasta `common` contém componentes reutilizáveis por toda a aplicação.

## Estrutura

### constants/

Constantes globais da aplicação.

---

### decorators/

Decorators personalizados reutilizáveis.

Exemplos:

- `@CurrentUser()`
- `@Roles()`

---

### dto/

DTOs reutilizáveis.

Exemplos:

- PaginationDto
- SearchDto
- FilterDto

---

### enums/

Enumerações globais.

---

### exceptions/

Exceções personalizadas.

---

### filters/

Exception Filters globais.

---

### interceptors/

Interceptors globais.

---

### interfaces/

Interfaces partilhadas.

---

### types/

Tipos TypeScript reutilizáveis.

---

### utils/

Funções utilitárias.

---

### validators/

Validadores personalizados com `class-validator`.
