# 🎨 Color Palette

## Introdução

Esta paleta define as cores oficiais utilizadas no APCM.

Todas as novas funcionalidades deverão utilizar exclusivamente estas cores.

---

## Primary

Atlantic Blue

HEX: #0F4C81

RGB: 15, 76, 129

Utilização:

- Sidebar
- Links
- Títulos
- Botões secundários

---

## Primary Dark

#163A5F

Utilização:

- Hover
- Header
- Elementos ativos
